# Llama Intelligence: Modular Web Dashboard

Este módulo proporciona una interfaz web premium para interactuar con Llama 3.1.

## Características
- **Carga de Archivos**: Sube archivos `.txt` o `.pdf` directamente desde el navegador.
- **Resumen en Tiempo Real**: Visualiza el progreso y los resultados de Llama.
- **Diseño Premium**: Interfaz moderna con Modo Oscuro y Glassmorphism.

## Cómo Ejecutar
1. Instalar dependencias: `pip install flask` (ya incluido en requirements.txt).
2. Ir a la carpeta: `cd web_dashboard`.
3. Ejecutar el servidor: `python app.py`.
4. Abrir en el navegador: `http://127.0.0.1:5000`.

---
*Nota: Asegúrate de que Ollama esté corriendo antes de iniciar el servidor.*
